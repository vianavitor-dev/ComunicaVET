O **ComunicaVet** é uma plataforma web desenvolvida para ajudar tutores de animais a encontrar clínicas veterinárias confiáveis próximas à sua localização.

O sistema combina **geolocalização, avaliações verificadas, correspondência por especialidade e recomendações baseadas em IA**, facilitando a escolha de atendimento veterinário de forma rápida e segura.

Este projeto foi desenvolvido como parte de um projeto acadêmico com o objetivo de aplicar tecnologias modernas de desenvolvimento web e explorar o uso de inteligência artificial no apoio ao cuidado com animais.

---

# Funcionalidades

- 🔍 Busca por clínicas veterinárias com base na localização do usuário  
- 🗺️ Mapa interativo com clínicas próximas  
- 🏥 Perfis de clínicas com informações e avaliações  
- ⚙️ Filtros por especialidade e serviços  
- 🧠 Recomendações e dicas de cuidado para pets baseadas em IA  

---

# Arquitetura

O ComunicaVet segue uma arquitetura cliente-servidor:

Frontend (React)<br>
↓<br>
API REST<br>
Backend (Spring Boot)<br>
↓          ↘ <br>
↓          Microserviço de IA (Python)<br>
↓ <br>
MySQL<br>


Essa separação facilita **manutenção, escalabilidade e organização do sistema**.
